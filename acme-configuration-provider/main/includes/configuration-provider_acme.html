Configuration property fixed at build time - All other configuration properties are overridable at runtime

Configuration property

Type

Default

acme.devservices.enabled

Enable or disable Dev Services explicitly. Dev Services are automatically enabled unless acme.environment.url is set.

Environment variable: ACME_DEVSERVICES_ENABLED

boolean

true

acme.devservices.image-name

The Acme configuration value provider container image to use. <p>

Details about default image on Docker Repository on Quay

Environment variable: ACME_DEVSERVICES_IMAGE_NAME

string

quay.io/jtama/acme-provider:native

acme.strict.rest

Should the extension be strict when correcting response headers.

If set to true all response will have added headers, otherwise a simple warning will be displayed at application startup.

Environment variable: ACME_STRICT_REST

boolean

false

acme.strict.utils

Should the extension be strict when correcting utils classes.

If set to true all targeted class will be augmented, otherwise a simple warning will be displayed at application startup.

Environment variable: ACME_STRICT_UTILS

boolean

false

acme.environment.url

The environment provider server URL.

Value must be a valid URI

Environment variable: ACME_ENVIRONMENT_URL

URI

required
